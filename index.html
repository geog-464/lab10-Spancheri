Q2: What is happening in the styleAll function? Explain what the styles variable is and what it contains, what the conditional statement is doing and what gets returned by the function.
The 'styleAll' function takes the features you pass through it (in this case, our placemarks) and stylizes them. It can change the colour, size, zoom level, opacity, etc. of the placemark. The variable 'styles' shows object-key-value pairs that define the style elements and puts them at default settings (to be changed with dot notation within the if statement)

Q5:What is the fundamental difference between layer.options.fill and layer.getLatLng(), beyond the fact that they return different pieces of information?
The fundamental difference, beyond the fact that 'layer.options.fill' returns a Boolean and 'layer.getLatLng()' returns an object,is that
layer.options.fill: a path class that contains option/constants shared between vector overlays. The fill method determines whether or not to fill the path with colour (T/F)- putting it to false disables it from adding colour to a path.
layer.getLatLng(): this method returns the latitude and longitude (Current geographical position) of a marker. It directly applies to markers, which are clickable/draggable icons on a map.
